Algoritimos Ordenação

Algoritmo de ordenação é um conjunto instruções para ordenar um array. Podemos também ordenar mais do que tipos númericos, podemos ordenar qualquer tipo de dado, se implementarmos uma codição de comparação ao tipo.

Propriedades Importantes

Algoritmo Estável
Preserva a ordem dos elementos
Algoritmos in-place
Não usa memória extra

Dividimos os Algoritmos de Ordenação por:

  • Por comparação
    • Simples
    • Dividir para conquistar
  • Em tempo linar
    • Preço a pagar (Memória)

BubbleSort

  • Baseado na troca de elementos adjacentes
  • Cada iteração leva o elemento maior para o final do vetor
